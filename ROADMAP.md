## 🔄 Processo de Priorização

### Matriz de Decisão

```
                    IMPACTO
                 Alto    Baixo
           ┌─────────┬─────────┐
    Alto   │ FAZER   │ PLANEJAR│
ESFORÇO    │ AGORA   │         │
           ├─────────┼─────────┤
    Baixo  │ QUICK   │ EVITAR  │
           │ WIN     │         │
           └─────────┴─────────┘
```

### Critérios de Priorização

1. **Impacto no usuário** (1-5)
2. **Impacto no negócio** (1-5)
3. **Esforço técnico** (1-5, inverso)
4. **Risco** (1-5, inverso)
5. **Dependências** (blocker ou não)

**Score = (Impacto Usuário × 2 + Impacto Negócio × 2 + Esforço + Risco) / 6**
